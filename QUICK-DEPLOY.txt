STEP 1: Download Theme from GitHub

STEP 2: Create config.php

In the extracted folder:

2. Rename: config.php
3. Edit with these values:

   define('CUSTOM_LOGIN_SLUG', 'your-secret-slug-2024');

STEP 3: Upload to Hostinger
----------------------------
1. Login to Hostinger dashboard
2. Click "File Manager"
3. Navigate to: public_html/wp-content/themes/
4. Delete old "shivendra-blog" folder (if exists)
5. Upload the entire "shivendra-blog" folder
   (or upload as ZIP and extract there)


STEP 4: Enable Custom Login
----------------------------
In File Manager:
1. Open: shivendra-blog/functions.php
2. Find line 163
3. Remove "//" from:
   // require_once SHIVENDRA_THEME_DIR . '/inc/custom-login.php';

   Should become:
   require_once SHIVENDRA_THEME_DIR . '/inc/custom-login.php';
4. Save


STEP 5: Activate Theme
-----------------------
1. Go to: yourdomain.com/wp-admin
2. Appearance â†’ Themes
3. Activate "Shivendra Blog"


STEP 6: Flush Permalinks
-------------------------
1. Settings â†’ Permalinks
2. Click "Save Changes" (don't change anything)


STEP 7: Test Custom Login
--------------------------
1. Save this URL: yourdomain.com/your-secret-slug-2024
2. Logout from WordPress
3. Visit the custom URL
